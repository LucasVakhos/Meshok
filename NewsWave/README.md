# NewsWave

Веб-версия NewsMaker на ASP.NET Core и DevExtreme.

## Что повторено из NewsMaker

- экран «Установки» с настройками запуска, BridgeNote и почтового сервера;
- экран «Отчёт» с командами «Начать рассылку» / «Остановить» и «Отправить мне»;
- ручной и автоматический запуск по дню недели и времени;
- процедуры chk_news и chk_subscribers;
- формирование Excel из представления new_prix_list;
- адресный буфер v_ss_buffer;
- персональный текст, ссылка отписки и архив новинок;
- удаление успешно отправленной записи из subscribers_send_buffer;
- интервалы в subscribers_send_setting.

Рассылка и отправка отчёта выполняются через настроенный SMTP вашего домена.
Подписчики читаются из subscribers только по полю email. Ручное добавление выполняет
INSERT INTO subscribers (email) VALUES (...) и не придумывает значения других полей.

## Запуск

В каталоге NewsWave выполните npm install, затем dotnet run.

DevExtreme 25.2.7 устанавливается локально командой npm install.
Настройки, введённые на странице NewsMaker, сохраняются в
App_Data/newsmaker-settings.json; каталог исключён из Git.

До настройки BridgeNote и SMTP запуск рассылки завершится безопасной ошибкой,
не выполняя отправку.
