# Meshok

Набор утилит для автоматизации работы с торговой площадкой Мешок.

## Состав

| Проект | Описание |
|--------|----------|
| `GH.Components` | Библиотека общих компонентов (гриды, конфигурации, атрибуты) |
| `LB.Libs` | Библиотека инфраструктуры (IniFile, шифрование, DevExpress-расширения) |
| `MehokBrowser` | Браузер-клиент для работы с площадкой Мешок |
| `NewsMaker` | Инструмент email-рассылки новостей по клиентам |
| `NewsWave` | Веб-приложение (ASP.NET Core + DevExtreme) |
| **`CsvToExcel`** | Конвертер CSV → Excel (.xlsx) |

## CsvToExcel — установка

1. Скачайте `CsvToExcel_Setup_x.x.x.exe` со страницы [Releases](https://github.com/LucasVakhos/Meshok/releases)
2. Запустите инсталлятор и следуйте инструкциям мастера
3. После установки приложение доступно через меню **Пуск** и ярлык на **рабочем столе**
4. Выберите CSV-файл — приложение предложит сохранить его как `.xlsx`

### Сборка из исходников

```powershell
# Требуется .NET 8 SDK и Inno Setup 6
cd CsvToExcel
.\build-installer.ps1
```
